# M-DAFTO: Multi-Stage Deferred Acceptance Based Fair Task Offloading in IoT-Fog Systems

This repository contains the simulation codebase for **M-DAFTO** (Multi-Stage Deferred Acceptance Based Fair Task Offloading) in IoT-Fog Systems, focusing on weight generation, urgency calculation, and preference ranking for fair task offloading.

---

## 🚀 Overview

In IoT-Fog computing architectures, tasks generated by IoT devices must be offloaded to appropriate Fog nodes. The allocation must balance delay, energy consumption, urgency, and resource fairness. 

This simulation:
1. **Generates simulation configurations**: Deploys a set of Fog networks with distinct capacities and generates high-frequency computational tasks with varying deadlines, phases (Pre/Post-surgery), and severity levels (Major/Minor).
2. **Computes offloading metrics**: Calculates the transmission delay, execution delay, return delay, and total energy consumed per task across all Fog nodes.
3. **Normalizes metrics globally**: Applies Min-Max normalization to delay and energy parameters.
4. **Calculates urgency values**: Employs multi-objective weight parameters to compute the urgency of each task relative to each Fog node, prioritizing critical/major tasks.
5. **Generates preference rankings**: Ranks tasks based on urgency for each Fog node.

## ⚙️ Mathematical Model

### 1. Delay & Energy Calculations
For each task $T_i$ on Fog node $F_j$:
* **Transmission Time**: 
  $$T_{\text{tx}} = \frac{\text{Input Size}}{\text{Uplink Data Rate}}$$
* **Execution Time**: 
  $$T_{\text{exec}} = \frac{\text{CPU Demanded}}{\text{VRU Capacity}}$$
* **Receiving Time**: 
  $$T_{\text{rx}} = \frac{\text{Output Size}}{\text{Downlink Data Rate}}$$
* **Offloading Delay**: 
  $$D_{ij} = T_{\text{tx}} + T_{\text{exec}} + T_{\text{rx}}$$
* **Energy**: 
  $$E_{ij} = (P_{\text{tx}} \times T_{\text{tx}}) + (P_{\text{exec}} \times T_{\text{exec}}) + (P_{\text{rx}} \times T_{\text{rx}})$$

### 2. Normalization
Global Min-Max normalization maps values to $[0, 1]$:
$$X_{\text{norm}} = \frac{X - X_{\text{min}}}{X_{\text{max}} - X_{\text{min}}}$$
*Note: Minor tasks apply a $0.5$ reduction weight to their normalized sums.*

### 3. Task Urgency
The urgency $U_{ij}$ of task $T_i$ for Fog node $F_j$ is calculated as:
$$U_{ij} = w_1 \cdot \frac{1}{\text{Deadline} - D_{ij}} + w_2 \cdot \frac{1}{\text{pref}(T_i)} + w_3 \cdot \frac{1}{E_{ij}} + w_4 \cdot \gamma_i$$
Where:
* $\text{pref}(T_i)$ is the number of Fog nodes that can complete $T_i$ before its deadline.
* $\gamma_i$ is the severity score ($1.5$ for Major, $1.0$ for Minor tasks).
